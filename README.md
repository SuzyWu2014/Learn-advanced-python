# Learn-advanced-python

就目前接触到的语言而言，Python算是我非常喜欢的一门语言了。语法简洁，却又可以清晰表达出复杂的算法逻辑，同时也采纳了一些其他语言的优势，例如 Functional programming 的 lazyness. 这本 git book 算是一本我个人的 python 进阶学习笔记，也可以说是个人的 Python Wiki，便于今后搜索查看。
